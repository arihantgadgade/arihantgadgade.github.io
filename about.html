Website Structure

Hosting this website on github pages has numerous benefits: cost, version control, everything is open source at this github repo for anyone to use.

My website home has a directory-like minimalist structure that I very much enjoy. I adopted it from Longest Levers

Also articles within sections are generally ordered chronologically from oldest to newest. I prefer that to generally how blogs/substacks show the newest writings. I like building up the writer/creator in my head from their oldest to newest writings.
